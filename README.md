### 📓 프로젝트 개요

- 설명 : 예금 상품 조회 및 추천 서비스
- 특징 :

<br/>

### 🦝 프로젝트 특징

- 유저 MBTI에 기반하여 알맞은 상품을 추천함
- 유저 경험(UX)을 우선으로 하여 로그인 없이 대부분의 서비스 제공
 

<br/>

### ⚙ 주요 기능

- 실시간 환율 계산기
- 지도에서 은행 지점 정보 제공
- MBTI 성격 유형에 따른 예금 상품 추천 알고리즘
- 시중 은행의 예금 상품 및 옵션에 따른 금리 조회
- 관심있는 상품과 옵션 모아보기 및 비교 차트 제공
- 다른 사용자와 금융 이야기를 나눌 수 있는 커뮤니티 게시판

<br/>

### ⌚ 프로젝트 진행 기간
2023.11.15(수) ~ 2023.11.24(금)

<br/>

## 🦾 팀 소개 
- **김상훈** : 팀장, Django를 활용한 BackEnd 개발 담당, ERD 설계, REST API 개발, 코드 품질 관리 및 리팩토링, 추천 알고리즘 개발, BackEnd 배포

- **김지연** : Vue.js 를 활용한 FrontEnd 개발 담당, UI 디자인, 문서정리, 지도 검색 구현, FrontEnd 배포, 서비스 테스트 및 디버깅
